The game is pre-packaged as a native Windows application. You do **not** need to install Java or configure any environment variables to play it.

### Step-by-Step Guide:

1. Click on the **[Releases](https://github.com/Louiechops101/Capstone-4-project/releases/tag/Game)** section on the right side of this GitHub page.
2. Under the latest version tag, click on **`FlappyBirdFX.zip`** under the "Assets" dropdown to download the game bundle.
3. Once downloaded, locate the `.zip` file in your system's `Downloads` folder.
4. **Extract (Unzip) the file completely.** * *Note: Do not try to run the game directly from inside the zipped archive, or the assets will fail to load.*
